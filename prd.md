## 1. 개요

### 1.1 프로젝트 요약

정부정책자금(저금리 대출, 보조금, 지원사업 등)을 소개하고, 이메일 및 카카오톡 채널을 통한 리드 수집을 목표로 하는 **원스크롤(One-Scroll) 랜딩페이지**를 기획·개발한다.

### 1.2 목적

| 구분          | 내용                                                                           |
| ------------- | ------------------------------------------------------------------------------ |
| 비즈니스 목표 | 정부정책자금 대출에 관심 있는 **중소기업, 소상공인, 예비창업자**를 리드로 확보 |
| 마케팅 목표   | 방문자 → 구독자 → 상담전환 퍼널 구축, 광고 캠페인 연동 시 **ROAS 극대화**      |
| 사용자 목표   | 복잡한 정부자금 정보를 **쉽고 빠르게** 파악하고, 간편하게 상담 신청            |

### 1.3 기대효과

- 짧은 시간 내 방문자 → 구독자 → 상담전환 달성
- 광고(메타, 구글, 네이버) 캠페인 연동 시 ROAS 상승
- 카카오톡 채널 구독자 확보를 통한 리마케팅 채널 구축
- 이메일 리스트 확보를 통한 장기 너처링(Nurturing) 가능

### 1.4 대상 사용자 (타겟 페르소나)

| 페르소나                    | 특징                                             | 니즈                                 |
| --------------------------- | ------------------------------------------------ | ------------------------------------ |
| **중소기업 대표** (40~55세) | 운전자금·시설자금 필요, 은행 대출 한도 부족 경험 | 저금리 정책자금 정보, 자격 요건 확인 |
| **소상공인** (30~50세)      | 카페·음식점·소매업 등 운영, 자금 유동성 고민     | 간편한 신청 절차, 빠른 상담 연결     |
| **예비창업자** (25~40세)    | 창업 준비 중, 초기 자금 확보 필요                | 창업자금 종류, 자격 조건, 신청 방법  |

## 2. 페이지 구조 (원스크롤 섹션 구성)

전체 페이지는 **단일 페이지(Single Page)**, 세로 스크롤 방식으로 아래 섹션 순서대로 구성한다.

```
┌─────────────────────────────────────┐
│  [S1] 히어로 섹션                     │
│  - 메인 카피 + 서브 카피              │
│  - CTA 버튼 (상담 신청)              │
├─────────────────────────────────────┤
│  [S2] 문제 제기 / 공감 섹션           │
│  - 타겟의 Pain Point 나열            │
├─────────────────────────────────────┤
│  [S3] 정부정책자금 소개               │
│  - 주요 자금 종류 카드형 소개          │
├─────────────────────────────────────┤
│  [S4] 지원 대상 / 자격 요건           │
│  - 체크리스트 형태                    │
├─────────────────────────────────────┤
│  [S5] 신청 프로세스                   │
│  - 3~4단계 스텝 인포그래픽            │
├─────────────────────────────────────┤
│  [S6] 성공 사례 / 후기               │
│  - 카드형 후기 캐러셀                 │
├─────────────────────────────────────┤
│  [S7] 신뢰 요소                      │
│  - 수치 실적, 인증 배지, 파트너 로고   │
├─────────────────────────────────────┤
│  [S8] FAQ                           │
│  - 아코디언 형태 5~7개               │
├─────────────────────────────────────┤
│  [S9] 리드 수집 폼 (CTA 섹션)        │
│  - 이메일 입력 + 카카오톡 채널 추가    │
├─────────────────────────────────────┤
│  [F] 푸터                            │
│  - 회사 정보, 개인정보처리방침 링크     │
└─────────────────────────────────────┘
```

## 3. 섹션별 상세 기획

### S1. 히어로 섹션 (Hero)

| 항목          | 내용                                                           |
| ------------- | -------------------------------------------------------------- |
| **목적**      | 3초 내 관심 유도, 핵심 가치 전달                               |
| **메인 카피** | "2025년 정부정책자금, 최대 3억까지 연 1%대 저금리" (예시)      |
| **서브 카피** | "중소기업·소상공인·예비창업자를 위한 맞춤형 자금 안내"         |
| **CTA 버튼**  | `무료 자격 진단 받기` (주요 CTA, 클릭 시 S9 폼으로 스크롤)     |
| **비주얼**    | 신뢰감 있는 배경 이미지 또는 일러스트 (사무실, 성장 그래프 등) |
| **배지**      | "2025년 최신 기준 반영", "무료 상담" 등 소형 배지              |

**디자인 노트:**

- 모바일 퍼스트: 히어로 영역이 뷰포트 전체를 차지
- CTA 버튼은 고대비 색상 (예: 파란색 배경 + 흰색 텍스트)
- 스크롤 유도 화살표 하단 배치

### S2. 문제 제기 / 공감 섹션

| 항목     | 내용                                   |
| -------- | -------------------------------------- |
| **목적** | 타겟의 Pain Point를 언급하여 공감 형성 |
| **구성** | 아이콘 + 짧은 문장 3~4개               |

**Pain Point 예시:**

- 😩 "은행 대출 금리가 너무 높아 부담스럽다"
- 😩 "정부자금이 있다는 건 알지만 어디서 어떻게 신청하는지 모르겠다"
- 😩 "서류 준비가 복잡해서 포기한 적 있다"
- 😩 "자격 요건에 해당하는지 확인하기 어렵다"

**디자인 노트:**

- 회색/연한 배경으로 히어로와 시각적 분리
- 아이콘은 미니멀 라인 스타일

### S3. 정부정책자금 소개

| 항목     | 내용                             |
| -------- | -------------------------------- |
| **목적** | 주요 정책자금 종류를 한눈에 소개 |
| **구성** | 카드형 레이아웃 (3~4종)          |

**카드 구성 예시:**

| 카드 | 자금명                                | 한줄 설명              | 주요 조건               |
| ---- | ------------------------------------- | ---------------------- | ----------------------- |
| 1    | 중소벤처기업진흥공단 정책자금         | 중소기업 운전·시설자금 | 최대 100억, 연 2%대     |
| 2    | 소상공인시장진흥공단 직접대출         | 소상공인 운영자금      | 최대 7천만원, 연 2~3%   |
| 3    | 신용보증기금 보증부 대출              | 신용 보완을 통한 대출  | 보증비율 최대 95%       |
| 4    | 예비창업자 지원 (청년창업사관학교 등) | 창업 초기 자금 지원    | 최대 1억, 무이자~저금리 |

**디자인 노트:**

- 모바일: 세로 스택 / 데스크톱: 2×2 그리드
- 각 카드에 `자세히 보기 →` 텍스트 링크 (상담 폼으로 이동)

### S4. 지원 대상 / 자격 요건

| 항목     | 내용                         |
| -------- | ---------------------------- |
| **목적** | "나도 해당되나?" 궁금증 해소 |
| **구성** | 체크리스트 형태              |

**체크리스트 예시:**

- ✅ 사업자등록증 보유 중소기업 또는 소상공인
- ✅ 업력 7년 이내 기업 (일부 자금은 업력 무관)
- ✅ 국세·지방세 체납 없음
- ✅ 신용등급 일정 기준 이상 (자금별 상이)
- ✅ 예비창업자 (사업자등록 전이라도 가능한 자금 존재)

**하단 CTA:**

> "해당 여부가 확실하지 않으신가요? → **무료 자격 진단 받기**"

### S5. 신청 프로세스

| 항목     | 내용                                                        |
| -------- | ----------------------------------------------------------- |
| **목적** | 복잡해 보이는 절차를 시각적으로 간소화하여 심리적 허들 낮춤 |
| **구성** | 3~4단계 스텝 인포그래픽                                     |

**스텝 구성:**

```
[STEP 1]           [STEP 2]           [STEP 3]           [STEP 4]
 무료 상담 신청  →   자격 진단 &     →   서류 준비 &     →   자금 승인 &
                    맞춤 자금 안내       신청 대행 지원       입금 완료
```

**디자인 노트:**

- 수평 타임라인 (데스크톱) / 수직 타임라인 (모바일)
- 각 스텝에 아이콘 + 짧은 설명 텍스트
- 현재 단계(STEP 1)를 강조 색상으로 표시하여 "지금 바로 시작" 느낌 부여

### S6. 성공 사례 / 후기

| 항목     | 내용                                    |
| -------- | --------------------------------------- |
| **목적** | 사회적 증거(Social Proof)로 신뢰도 상승 |
| **구성** | 후기 카드 캐러셀 (3~5개)                |

**후기 카드 구성:**

- 프로필 이미지 (또는 이니셜 아바타)
- 이름 + 업종 (예: "김OO 대표 / 제조업")
- 후기 내용 (2~3줄)
- 승인 금액 + 금리 (예: "2억 승인, 연 1.9%")
- 별점 (★★★★★)

**디자인 노트:**

- 자동 슬라이드 + 수동 스와이프 지원
- 모바일: 1장씩 / 데스크톱: 3장 동시 노출

### S7. 신뢰 요소 (Trust Signals)

| 항목     | 내용                                |
| -------- | ----------------------------------- |
| **목적** | 서비스 신뢰도와 전문성 입증         |
| **구성** | 수치 실적 + 인증 배지 + 파트너 로고 |

**수치 실적 (카운트업 애니메이션):**

- 📊 누적 상담 건수: **12,000건+**
- 💰 누적 승인 금액: **1,500억+**
- ⭐ 고객 만족도: **98%**
- 👥 전문 컨설턴트: **30명+**

**파트너/인증:**

- 중소벤처기업부, 소상공인시장진흥공단, 신용보증기금, 기술보증기금 로고
- 기업 인증 배지 (ISO, 공식 대리인 자격 등)

### S9. 리드 수집 폼 (CTA 섹션)

| 항목          | 내용                                                  |
| ------------- | ----------------------------------------------------- |
| **목적**      | 핵심 전환 구간. 이메일 수집 + 카카오톡 채널 추가      |
| **헤드라인**  | "지금 무료 자격 진단을 받아보세요"                    |
| **서브 카피** | "30초면 충분합니다. 맞춤 정책자금을 안내해 드립니다." |

**폼 필드:**

| 필드               | 타입     | 필수 여부 | 설명                                        |
| ------------------ | -------- | --------- | ------------------------------------------- |
| 이름               | text     | 필수      | 2글자 이상                                  |
| 연락처             | tel      | 필수      | 010-XXXX-XXXX 형식                          |
| 이메일             | email    | 필수      | 유효성 검증                                 |
| 업종               | select   | 선택      | 제조/서비스/요식/IT/기타                    |
| 희망 자금 규모     | select   | 선택      | 3천만원 이하 / 3천~1억 / 1억~3억 / 3억 이상 |
| 개인정보 수집 동의 | checkbox | 필수      | 개인정보처리방침 링크 포함                  |
| 마케팅 수신 동의   | checkbox | 선택      | 이메일/SMS 수신 동의                        |

**제출 버튼:** `무료 자격 진단 신청하기`

**카카오톡 채널 추가:**

- 폼 하단 또는 옆에 `카카오톡 채널 추가하기` 버튼 배치
- "카카오톡으로 빠른 상담을 받으세요" 안내 문구
- 카카오톡 채널 추가 API 연동

**제출 후 동작:**

1. 성공 시: Thank You 모달 또는 페이지 전환 → "신청이 완료되었습니다. 24시간 내 연락드리겠습니다."
2. GA4 전환 이벤트 발생 (`generate_lead`)
3. Meta Pixel / Google Ads 전환 추적 코드 실행
4. 리드 데이터 → CRM(구글 시트, Notion, HubSpot 등)으로 자동 전송
